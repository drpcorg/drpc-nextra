## Accounts info

Retrieve key data about accounts on the Mantle network.

[eth_accounts](/mantle-api/accountinfo/eth_accounts) : Retrieves a list of accounts controlled by the connected client. \
[eth_getBalance](/mantle-api/accountinfo/eth_getBalance) : Returns the balance of an account. \
[eth_getCode](/mantle-api/accountinfo/eth_getCode) : Fetches the contract code at a given address. \
[eth_getProof](/mantle-api/accountinfo/eth_getProof) : Provides a Merkle proof for an accountâ€™s state. \
[eth_getStorageAt](/mantle-api/accountinfo/eth_getStorageAt) : Retrieves stored data at a specific slot in the account's storage.

These methods provide essential account and contract data for interacting with Mantle.
