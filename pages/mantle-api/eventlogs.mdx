## Event logs

Provides methods for filtering and retrieving event logs.

[eth_getLogs](/mantle-api/eventlogs/eth_getLogs) : Retrieves logs from smart contracts based on filter criteria. \
[eth_newFilter](/mantle-api/eventlogs/eth_newFilter) : Creates a filter for tracking specific events. \
[eth_getFilterChanges](/mantle-api/eventlogs/eth_getFilterChanges) : Retrieves changes for an active filter since its last check. \
[eth_uninstallFilter](/mantle-api/eventlogs/eth_uninstallFilter) : Uninstalls an active filter. \
[eth_getFilterLogs](/mantle-api/eventlogs/eth_getFilterLogs) : Fetches logs from a previously created filter.

These methods are key for monitoring smart contract events on Mantle.
