## Chain info


Provides information about the BSC   network, including its protocol version, configuration, and current status.

[eth_chainId](/bsc-api/chaininfo/eth_chainId) : Returns the chain ID of the network.  \
[eth_protocolVersion](/bsc-api/chaininfo/eth_protocolVersion) : Provides the protocol version in use by the client.  \
[net_listening](/bsc-api/chaininfo/net_listening) : Checks if the client is listening for network connections. \
[net_version](/bsc-api/chaininfo/net_version) : Retrieves the current network version.  \
[net_peerCount](/bsc-api/chaininfo/net_peerCount) : Returns the number of active peers connected to the node.  \
[eth_syncing](/bsc-api/chaininfo/eth_syncing) : Indicates whether the node is synchronizing with the network.  \
[eth_hashrate](/bsc-api/chaininfo/eth_hashrate) : Provides the node's current mining hashrate.

These methods help monitor the health, configuration, and performance of the network.
