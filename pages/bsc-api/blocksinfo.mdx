## Blocks info

Retrieves detailed information from a specified block, including transactions, receipts, and block data. These methods provide insights into block details, transaction counts, and execution outcomes.

[eth_blockNumber](/bsc-api/blocksinfo/eth_blockNumber) : Gets the most recent block number. \
[eth_getBlockByHash](/bsc-api/blocksinfo/eth_getBlockByHash) : Retrieves block details using its hash. \
[eth_getBlockByHash#full](/bsc-api/blocksinfo/eth_getBlockByHashfull) : Returns complete block details using its hash. \
[eth_getBlockByNumber](/bsc-api/blocksinfo/eth_getBlockByNumber) : Retrieves block details using its number. \
[eth_getBlockByNumber#full](/bsc-api/blocksinfo/eth_getBlockByNumberfull) : Fetches full block details using its number. \
[eth_newBlockFilter](/bsc-api/blocksinfo/eth_newBlockFilter) : Available only on paid tier. Sets a filter to monitor new blocks. \
[eth_getBlockReceipts](/bsc-api/blocksinfo/eth_getBlockReceipts) : Gets transaction receipts for a specific block. \
[eth_getBlockTransactionCountByHash](/bsc-api/blocksinfo/eth_getBlockTransactionCountByHash) : Returns the number of transactions in a block by its hash. \
[eth_getBlockTransactionCountByNumber](/bsc-api/blocksinfo/eth_getBlockTransactionCountByNumber) :  Retrieves the number of transactions in a block by its number.

These methods provide essential tools for tracking and analyzing blockchain activity.
