## Getting uncles

Retrieves information about "uncle" blocks, which are valid but not included in the main blockchain.

[eth_getUncleByBlockHashAndIndex](/bsc-api/gettinguncles/eth_getUncleByBlockHashAndIndex) : Retrieves uncle block information by block hash and index. \
[eth_getUncleByBlockNumberAndIndex](/bsc-api/gettinguncles/eth_getUncleByBlockNumberAndIndex) : Fetches uncle block details by block number and index. \
[eth_getUncleCountByBlockHash](/bsc-api/gettinguncles/eth_getUncleCountByBlockHash) : Returns the number of uncles for a specific block hash. \
[eth_getUncleCountByBlockNumber](/bsc-api/gettinguncles/eth_getUncleCountByBlockNumber) : Retrieves the number of uncles for a block by its number.

These methods provide key details for analyzing uncle blocks on the network.
