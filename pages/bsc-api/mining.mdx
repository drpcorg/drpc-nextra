## Mining

Provides details on the mining state of the node.

[eth_coinbase](/bsc-api/mining/eth_coinbase) : Returns the address of the miner receiving mining rewards. \
[eth_mining](/bsc-api/mining/eth_mining) : Indicates whether the node is actively mining blocks.

These methods are useful for monitoring the mining status of a node.
