## Subscriptions

Allows subscribing to specific events in real-time via WebSockets.

[eth_subscribe](/bsc-api/subscriptions/eth_subscribe) : Subscribes to blockchain events like new blocks, pending transactions, or logs. \
[eth_unsubscribe](/bsc-api/subscriptions/eth_unsubscribe) : Unsubscribes from an existing event subscription.

These methods are essential for real-time event tracking and monitoring on the network.
