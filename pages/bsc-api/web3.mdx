## Web3

Returns general network configuration data for nodes.

[web3_clientVersion](/bsc-api/web3/web3_clientVersion) : Returns the version of the node client in use. \
[web3_sha3](/bsc-api/web3/web3_sha3) : Computes the Keccak-256 (SHA3) hash of the provided data.

These methods are essential for node management and cryptographic operations on the network.
