## Event logs

Provides logs of events emitted by smart contracts, helping track token transfers and state changes.

[eth_getLogs](/bsc-api/eventlogs/eth_getLogs) : Fetches logs based on specific filter criteria. \
[eth_newFilter](/bsc-api/eventlogs/eth_newFilter) : Available only on paid tier. Creates a new filter to track events. \
[eth_getFilterChanges](/bsc-api/eventlogs/eth_getFilterChanges) : Available only on paid tier. Retrieves updates for an active filter since the last query. \
[eth_uninstallFilter](/bsc-api/eventlogs/eth_uninstallFilter) : Removes an active filter. \
[eth_getFilterLogs](/bsc-api/eventlogs/eth_getFilterLogs) : Available only on paid tier. Retrieves logs from an existing filter.

These methods are vital for tracking smart contract activity.
