## Executing transactions

Executing smart contracts, and performing on-chain operations.

[eth_call](/bsc-api/executingtransactions/eth_call) : Executes a read-only call to a smart contract without altering state. \
[eth_sendRawTransaction](/bsc-api/executingtransactions/eth_sendRawTransaction) : Sends a raw, signed transaction to the blockchain for execution.

These methods are essential for interacting with smart contracts and sending transactions on the network.
