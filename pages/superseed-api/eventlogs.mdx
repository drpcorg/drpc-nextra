## Event logs

Allows the filtering and retrieval of event logs generated by smart contracts.

[eth_getLogs](/superseed-api/eventlogs/eth_getLogs) : Retrieves event logs based on specified filter criteria. \
[eth_newFilter](/superseed-api/eventlogs/eth_newFilter) : Available only on paid tier. Creates a new filter to monitor logs.  \
[eth_getFilterChanges](/superseed-api/eventlogs/eth_getFilterChanges) : Available only on paid tier. Retrieves changes for a filter since its last check. \
[eth_uninstallFilter](/superseed-api/eventlogs/eth_uninstallFilter) : Uninstalls a previously created filter.  \
[eth_getFilterLogs](/superseed-api/eventlogs/eth_getFilterLogs) : Available only on paid tier. Fetches logs matching a specific filter.

These methods help track and monitor contract events on the Superseed network.
