## Gas estimation

Provides tools to estimate gas fees and optimize transaction costs.

[eth_estimateGas](/superseed-api/gasestimation/eth_estimateGas) : Estimates the gas required for a transaction. \
[eth_gasPrice](/superseed-api/gasestimation/eth_gasPrice) : Returns the current gas price on the network. \
[eth_maxPriorityFeePerGas](/superseed-api/gasestimation/eth_maxPriorityFeePerGas) : Retrieves the maximum priority fee per gas unit.

These methods help optimize transaction costs by estimating the necessary gas fees on the Superseed network.
