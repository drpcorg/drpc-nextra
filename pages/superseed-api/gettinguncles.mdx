## Getting uncles

Provides methods to retrieve information about uncle blocks.

[eth_getUncleCountByBlockHash](/superseed-api/gettinguncles/eth_getUncleCountByBlockHash) : Retrieves the number of uncle blocks for a given block by its hash.
[eth_getUncleCountByBlockNumber](/superseed-api/gettinguncles/eth_getUncleCountByBlockNumber) : Returns the number of uncle blocks for a specified block number.

These methods allow developers to analyze uncle block occurrences on the Superseed network.
