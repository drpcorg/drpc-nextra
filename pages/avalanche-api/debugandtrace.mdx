## Debug and trace

Available only on the paid tier. These methods are useful for analyzing execution flows and debugging.

[debug_traceBlockByHash](/avalanche-api/debugandtrace/debug_traceBlockByHash) : Traces the execution of all transactions in a block by its hash. \
[debug_traceBlockByNumber](/avalanche-api/debugandtrace/debug_traceBlockByNumber) : Traces the execution of all transactions in a block by its number. \
[debug_traceTransaction](/avalanche-api/debugandtrace/debug_traceTransaction) : Provides detailed trace information for a specific transaction.  \
[debug_traceCall](/avalanche-api/debugandtrace/debug_traceCall) : Traces a specific call without altering the blockchain state.\

These methods provide advanced debugging capabilities on the Avalanche network.
