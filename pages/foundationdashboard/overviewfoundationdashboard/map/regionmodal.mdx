### Region modal

There are two types of this modal: for regions with active nodes and for regions with zero active nodes.

| **Empty Region**                                                                                                                                       | **Not Empty Region**                                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Active Nodes** — always 0                                                                                                                            | **Status, Active Nodes, Latency, 24h Served %** — everything as in the region’s card                                                                                                                                                                                                                                |
| **Latency** — Average latency over 30 minutes                                                                                                          | **Provider’s name** — shown for each provider in the region                                                                                                                                                                                                                                                         |
| **Requests from here** — Number of requests in the past 24 HOURS from this region                                                                      | **Status** — Provider status in this region for the last minute, converted to 4 color designations using the network status formula                                                                                                                                                                                 |
| **Share of requests, 24h** — The percentage of the number of requests in the past 24 HOURS from this region                                            | **Latency** — Average latency over 30 minutes, only requests from this region, processed by providers in this region (no cross-region requests), considering the list of allowed methods                                                                                                                            |
|                                                                                                                                                        | **Uptime** — Uptime based on provider statuses over the last 24 HOURS, calculated using the formula on the side                                                                                                                                                                                                     |
|                                                                                                                                                        | **24h req** — Number of requests in the past 24 HOURS at this provider (regardless of the outcome)                                                                                                                                                                                                                  |
