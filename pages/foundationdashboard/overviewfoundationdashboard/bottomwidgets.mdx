## Bottom widgets

Bottom Widgets show high-level network performance metrics.

### Latency by network

Average latency for the last 30 minutes in the network

### Total requests for 24h

The number of requests in the network for the last 24 hours

### Average req/sec for 24h

`Total requests for 24h / (60 * 60 * 24)`

### Monthly uptime

- for every 5-minute interval for the past 30 days we found the best status among all providers in this network
- we calculate uptime for this set of statuses using the formula above
