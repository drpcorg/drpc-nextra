# Arbitrum API overview

These endpoints allow to retrieve information about blocks, transactions, balances, logs, and more, facilitating efficient blockchain development and integration.

## Accounts info

Returns data about an account's balance, contract code, or stored data on the blockchain.

[eth_accounts](/arbitrum-api/accountinfo/eth_accounts) \
[eth_getBalance](/arbitrum-api/accountinfo/eth_getBalance) \
[eth_getCode](/arbitrum-api/accountinfo/eth_getCode) \
[eth_getProof](/arbitrum-api/accountinfo/eth_getProof) \
[eth_getStorageAt](/arbitrum-api/accountinfo/eth_getStorageAt)
