## Getting uncles

Retrieves information about uncle blocks (blocks rejected by the network).

[eth_getUncleByBlockHashAndIndex](/tron-api/gettinguncles/eth_getUncleByBlockHashAndIndex) : Fetches details of an uncle block by its hash and index. \
[eth_getUncleByBlockNumberAndIndex](/tron-api/gettinguncles/eth_getUncleByBlockNumberAndIndex) : Retrieves an uncle block by its block number and index. \
[eth_getUncleCountByBlockHash](/tron-api/gettinguncles/eth_getUncleCountByBlockHash) : Returns the count of uncles for a specific block hash. \
[eth_getUncleCountByBlockNumber](/tron-api/gettinguncles/eth_getUncleCountByBlockNumber) : Retrieves the number of uncles for a block number.

These methods are useful for tracking uncle blocks on Tron.
