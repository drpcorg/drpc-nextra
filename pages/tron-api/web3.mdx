## Web3

<Meta title="Tron web3 info | dRPC" />

Provides utility functions to interact with the blockchain.

[web3_clientVersion](/tron-api/web3/web3_clientVersion) : Returns the version of the client used by the node. \
[web3_sha3](/tron-api/web3/web3_sha3) : Computes the Keccak-256 (SHA3) hash of the provided input data.

These methods assist with node version management and cryptographic operations on the Tron network.
