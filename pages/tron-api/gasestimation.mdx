## Gas estimation

Estimates gas prices and consumption for transactions.

[eth_feeHistory](/tron-api/gasestimation/eth_feeHistory) : Retrieves historical gas prices over a range of blocks. \
[eth_estimateGas](/tron-api/gasestimation/eth_estimateGas) : Estimates the gas required to execute a transaction. \
[eth_gasPrice](/tron-api/gasestimation/eth_gasPrice) : Returns the current gas price on the network. \
[eth_createAccessList](/tron-api/gasestimation/eth_createAccessList) : Generates an access list for transaction optimization. \
[eth_maxPriorityFeePerGas](/tron-api/gasestimation/eth_maxPriorityFeePerGas) : Retrieves the maximum priority fee for gas.

These methods help optimize gas usage and costs on Tron.
