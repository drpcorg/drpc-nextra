## Subscriptions

Enables WebSocket connections for continuous monitoring of blockchain changes.

[eth_subscribe](/tron-api/subscriptions/eth_subscribe) : Subscribes to events such as new blocks, pending transactions, or logs. \
[eth_unsubscribe](/tron-api/subscriptions/eth_unsubscribe) : Unsubscribes from an active WebSocket subscription.

These methods allow for real-time event tracking on the Tron network.
