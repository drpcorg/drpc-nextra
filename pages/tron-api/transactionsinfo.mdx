## Transactions info

Accesses transaction details, such as their state, count, or receipts.

[eth_getTransactionByHash](/tron-api/transactionsinfo/eth_getTransactionByHash) : Fetches transaction details using its hash. \
[eth_getTransactionCount](/tron-api/transactionsinfo/eth_getTransactionCount) : Returns the number of transactions sent from an address. \
[eth_getTransactionReceipt](/tron-api/transactionsinfo/eth_getTransactionReceipt) : Retrieves the receipt of a processed transaction. \
[eth_newPendingTransactionFilter](/tron-api/transactionsinfo/eth_newPendingTransactionFilter) : Available only on paid tier. Sets up a filter to monitor pending transactions. \
[eth_getTransactionByBlockHashAndIndex](/tron-api/transactionsinfo/eth_getTransactionByBlockHashAndIndex) : Retrieves a transaction by block hash and index. \
[eth_getTransactionByBlockNumberAndIndex](/tron-api/transactionsinfo/eth_getTransactionByBlockNumberAndIndex) : Fetches a transaction by block number and index. \
[txpool_content](/tron-api/transactionsinfo/txpool_content) : Displays the contents of the transaction pool.

These methods assist in transaction management and tracking on Tron.
