## Transactions info

Enables access to transaction details, outputs, and unspent funds.

[getrawtransaction](/bitcoin-api/transactionsinfo/getrawtransaction): Retrieves the raw data of a transaction by ID. \
[gettransaction](/bitcoin-api/transactionsinfo/gettransaction): Provides detailed information about a specific transaction. \
[gettxout](/bitcoin-api/transactionsinfo/gettxout): Fetches details of an unspent transaction output. \
[listunspent](/bitcoin-api/transactionsinfo/listunspent): Lists unspent transaction outputs for addresses. \
[getrecievedbyaddress](/bitcoin-api/transactionsinfo/getrecievedbyaddress): Shows the total amount received by a specific address.

These methods facilitate precise transaction data access and management.
