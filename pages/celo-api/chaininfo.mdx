## Chain info

Supplies details about the network, its protocol version, and its configuration.

[eth_chainId](/celo-api/chaininfo/eth_chainId) : Retrieves the chain ID of the network.  \
[eth_protocolVersion](/celo-api/chaininfo/eth_protocolVersion) : Returns the protocol version used by the client. \
[net_listening](/celo-api/chaininfo/net_listening) : Checks if the client is actively listening for network connections. \
[net_version](/celo-api/chaininfo/net_version) : Returns the current network version.  \
[net_peerCount](/celo-api/chaininfo/net_peerCount) : Retrieves the number of active peers. \
[eth_syncing](/celo-api/chaininfo/eth_syncing) : Indicates the synchronization status of the node. \
[eth_hashrate](/celo-api/chaininfo/eth_hashrate) : Provides the current mining hashrate of the network.

These methods give valuable insight into the health, configuration, and operational status of the Celo network.
