# Accounts info

Provides key methods for retrieving various types of data related to accounts on the Celo network. It includes information about account balances, contract codes, and on-chain stored data.

[eth_accounts](/celo-api/accountinfo/eth_accounts) : Retrieves a list of accounts controlled by the connected client. \
[eth_getBalance](/celo-api/accountinfo/eth_getBalance) : Returns the balance of an account. \
[eth_getCode](/celo-api/accountinfo/eth_getCode) : Fetches the contract code stored at an address. \
[eth_getProof](/celo-api/accountinfo/eth_getProof) : Provides a Merkle proof for an accountâ€™s state. \
[eth_getStorageAt](/celo-api/accountinfo/eth_getStorageAt) : Returns data stored at a specific slot in the account's storage.

These methods enable to efficiently manage and retrieve vital information about accounts and smart contracts on the Celo network.
