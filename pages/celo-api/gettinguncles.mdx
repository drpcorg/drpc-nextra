## Getting uncles

Retrieves information about "uncle" blocks, which are valid but not included in the blockchain.

[eth_getUncleCountByBlockHash](/celo-api/gettinguncles/eth_getUncleCountByBlockHash) : Returns the number of uncle blocks for a block hash. \
[eth_getUncleCountByBlockNumber](/celo-api/gettinguncles/eth_getUncleCountByBlockNumber) : Returns the number of uncle blocks for a block number. \
[eth_getUncleByBlockHashAndIndex](/celo-api/gettinguncles/eth_getUncleByBlockHashAndIndex) : Returns uncle block by hash and index. \
[eth_getUncleByBlockNumberAndIndex](/celo-api/gettinguncles/eth_getUncleByBlockNumberAndIndex) : Returns uncle block by number and index.

These methods allow developers to analyze uncle block occurrences on the Celo network.
