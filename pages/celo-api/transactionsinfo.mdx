## Transactions info

Retrieves data on individual transactions, such as status, receipt, and inclusion in a specific block.

[eth_getTransactionByHash](/celo-api/transactionsinfo/eth_getTransactionByHash) : Fetches transaction details using its hash. \
[eth_getTransactionCount](/celo-api/transactionsinfo/eth_getTransactionCount) : Returns the number of transactions sent from an address. \
[eth_getTransactionReceipt](/celo-api/transactionsinfo/eth_getTransactionReceipt) : Retrieves the receipt of a processed transaction. \
[eth_newPendingTransactionFilter](/celo-api/transactionsinfo/eth_newPendingTransactionFilter) : Creates a filter for pending transactions. \
[eth_getTransactionByBlockHashAndIndex](/celo-api/transactionsinfo/eth_getTransactionByBlockHashAndIndex) : Retrieves a transaction by block hash and index. \
[eth_getTransactionByBlockNumberAndIndex](/celo-api/transactionsinfo/eth_getTransactionByBlockNumberAndIndex) : Retrieves a transaction by block number and index. \
[txpool_content](/celo-api/transactionsinfo/txpool_content) : Provides details of the transaction pool content.

These methods offer crucial data for tracking and managing transactions on the Celo network.
