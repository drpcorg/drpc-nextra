## Web3

Returns general network configuration data for nodes.

[web3_clientVersion](/celo-api/web3/web3_clientVersion) : Returns the version of the node client in use. \
[web3_sha3](/celo-api/web3/web3_sha3) : Computes the Keccak-256 (SHA3) hash of the provided data.

These methods are essential for managing node information and verifying data integrity on the Celo network.
