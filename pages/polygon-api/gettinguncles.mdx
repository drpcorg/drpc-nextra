## Getting uncles

Retrieves information about "uncle" blocks, which are valid but not included in the blockchain.

[eth_getUncleCountByBlockHash](/polygon-api/gettinguncles/eth_getUncleCountByBlockHash) : Returns the number of uncle blocks for a given block hash. \
[eth_getUncleCountByBlockNumber](/polygon-api/gettinguncles/eth_getUncleCountByBlockNumber) : Retrieves the number of uncle blocks for a specific block number.

These methods are useful for analyzing uncle blocks on the Polygon network.
