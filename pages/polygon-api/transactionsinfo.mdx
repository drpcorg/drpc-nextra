## Transactions info

Provides methods to retrieve transaction details, counts, and receipts.

[eth_getTransactionByHash](/polygon-api/transactionsinfo/eth_getTransactionByHash) : Retrieves details of a transaction by its hash.  \
[eth_getTransactionCount](/polygon-api/transactionsinfo/eth_getTransactionCount) : Returns the number of transactions sent from an address.  \
[eth_getTransactionReceipt](/polygon-api/transactionsinfo/eth_getTransactionReceipt) : Fetches the receipt of a processed transaction.  \
[eth_newPendingTransactionFilter](/polygon-api/transactionsinfo/eth_newPendingTransactionFilter) : Available only on paid tier. Sets up a filter for monitoring pending transactions. \
[eth_getTransactionByBlockHashAndIndex](/polygon-api/transactionsinfo/eth_getTransactionByBlockHashAndIndex) : Retrieves a transaction by block hash and index.  \
[eth_getTransactionByBlockNumberAndIndex](/polygon-api/transactionsinfo/eth_getTransactionByBlockNumberAndIndex) : Fetches a transaction by block number and index.  \
[txpool_content](/polygon-api/transactionsinfo/txpool_content) : Displays the contents of the transaction pool.

These methods assist in transaction management and tracking on the Polygon network.
