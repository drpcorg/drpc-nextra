## Event logs

Provides methods for filtering and retrieving event logs.

[eth_getLogs](/polygon-api/eventlogs/eth_getLogs) : Retrieves logs based on specified filter criteria.  \
[eth_newFilter](/polygon-api/eventlogs/eth_newFilter) : Creates a new filter for monitoring specific events. \
[eth_getFilterChanges](/polygon-api/eventlogs/eth_getFilterChanges) : Retrieves changes for an active filter since its last check.  \
[eth_uninstallFilter](/polygon-api/eventlogs/eth_uninstallFilter) : Uninstalls a specified filter.  \
[eth_getFilterLogs](/polygon-api/eventlogs/eth_getFilterLogs) : Fetches logs matching a filter.

These methods are essential for monitoring smart contract activities on the Polygon network.
