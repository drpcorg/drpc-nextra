## Gas estimation

Provides methods to estimate the gas required for transactions and fetch information related to gas prices.

[eth_feeHistory](/polygon-api/gasestimation/eth_feeHistory) : Retrieves historical gas prices over a specified range of blocks. \
[eth_estimateGas](/polygon-api/gasestimation/eth_estimateGas) : Estimates the gas needed for executing a transaction. \
[eth_gasPrice](/polygon-api/gasestimation/eth_gasPrice) : Returns the current gas price on the network. \
[eth_createAccessList](/polygon-api/gasestimation/eth_createAccessList) : Generates an access list to optimize gas usage. \
[eth_maxPriorityFeePerGas](/polygon-api/gasestimation/eth_maxPriorityFeePerGas) : Provides the maximum priority fee for gas.

These methods help optimize gas costs and efficiency on the Polygon network.
