## Mining

Contains methods related to mining operations.

[eth_coinbase](/polygon-api/mining/eth_coinbase) : Returns the address of the miner receiving rewards.  \
[eth_mining](/polygon-api/mining/eth_mining) : Indicates whether the node is currently mining new blocks.

These methods are essential for monitoring mining activities on the Polygon network.
