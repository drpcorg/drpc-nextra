## Subscriptions

Enables real-time tracking of specific Sonic events via WebSockets.

[eth_subscribe](/sonic-api/subscriptions/eth_subscribe): Subscribes to blockchain events, such as new blocks or logs. \
[eth_unsubscribe](/sonic-api/subscriptions/eth_unsubscribe): Cancels an existing subscription.

These methods are crucial for live event monitoring on the Sonic network.
