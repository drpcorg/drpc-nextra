## Chain info


Provides information about the Berachain network, including its protocol version, configuration, and current status.

[eth_chainId](/berachain-api/chaininfo/eth_chainId) : Obtains the unique chain identifier of the Berachain network. \
[eth_protocolVersion](/berachain-api/chaininfo/eth_protocolVersion) : Displays the version of the Ethereum protocol that the client is currently utilizing. \
[net_listening](/berachain-api/chaininfo/net_listening) : Checks whether the client is actively accepting incoming network connections. \
[net_version](/berachain-api/chaininfo/net_version) : Retrieves the current network ID of the Berachain network.  \
[net_peerCount](/berachain-api/chaininfo/net_peerCount) : Indicates the number of peers currently connected to the node.  \
[eth_syncing](/berachain-api/chaininfo/eth_syncing) : Shows whether the node is in the process of synchronizing with the Berachain blockchain.  \
[eth_hashrate](/berachain-api/chaininfo/eth_hashrate) : Provides the nodeâ€™s current mining hashrate.

These endpoints are crucial for overseeing the health, configuration, and performance of the Berachain network.
