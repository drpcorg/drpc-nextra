## Web3

Provides overall configuration details for nodes in the network.

[web3_clientVersion](/berachain-api/web3/web3_clientVersion) : Fetches the current version of the node client in use. \
[web3_sha3](/berachain-api/web3/web3_sha3) : Computes the Keccak-256 (SHA3) hash for the given input data.

These endpoints are key for configuring nodes and performing cryptographic operations within the Berachain network.
