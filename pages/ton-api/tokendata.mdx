## Token Data

Retrieve specific token-related data on Ton Chain.

[getTokenData](/ton-api/tokendata/getTokenData) Fetches metadata and details for a specific token, including its status and history. \
[nft/transfers](/ton-api/tokendata/nfttransfers): Retrieves a list of NFT transfer events. \
[nft/items](/ton-api/tokendata/nftitems): Provides information about individual NFTs, including their metadata, ownership. \
[nft/collections](/ton-api/tokendata/nftcollections): Retrieves data on NFT collections. \
[jetton/masters](/ton-api/tokendata/jettonmasters): Retrieves information about Jetton master contracts. \
[jetton/wallets](/ton-api/tokendata/jettonwallets): Provides details on Jetton wallets. \
[jetton/transfers](/ton-api/tokendata/jettontransfers): Retrieves a list of Jetton transfer events. \
[jetton/burns](/ton-api/tokendata/jettonburns): Provides information on Jetton burn events.

Focused on token-specific information, this group provides a way to access metadata and track tokens on Ton Chain, useful for applications involving tokenized assets.
