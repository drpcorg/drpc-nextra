## Account Info

Returns details about an account's balance, associated addresses, and wallet information on Ton Chain.

[getAddressInformation](/ton-api/accountinfo/getAddressInformation): Provides basic information for a specific address. \
[getExtendedAddressInformation](/ton-api/accountinfo/getExtendedAddressInformation): Offers extended information, including any additional metadata for the address. \
[getWalletInformation](/ton-api/accountinfo/getWalletInformation): Fetches details about the wallet type and state. \
[getAddressBalance](/ton-api/accountinfo/getAddressBalance): Returns the current balance of an address. \
[detectAddress](/ton-api/accountinfo/detectAddress): Detects and verifies the format of a given address. \
[wallet](/ton-api/accountinfo/wallet): Retrieves detailed information about a specific wallet.

This group provides essential tools for accessing account-related data, including balances and wallet details, enabling efficient management and verification of Ton Chain addresses.
