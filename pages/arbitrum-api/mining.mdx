## Mining

Provides details on the mining state of the node.

[eth_coinbase](/arbitrum-api/mining/eth_coinbase) : Returns the address of the miner who is receiving mining rewards. \
[eth_mining](/arbitrum-api/mining/eth_mining) : Indicates whether the node is actively mining blocks. \

These methods are useful for monitoring and managing mining activities on the Arbitrum network.
