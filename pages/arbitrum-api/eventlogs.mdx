## Event logs

Provides logs of events emitted by smart contracts, useful for tracking token transfers and state changes.

[eth_getLogs](/arbitrum-api/eventlogs/eth_getLogs) : Retrieves logs based on filter criteria. \
[eth_newFilter](/arbitrum-api/eventlogs/eth_newFilter) : Creates a new filter to monitor logs.\
[eth_getFilterChanges](/arbitrum-api/eventlogs/eth_getFilterChanges) : Retrieves changes for an active filter. \
[eth_uninstallFilter](/arbitrum-api/eventlogs/eth_uninstallFilter) : Uninstalls a specific filter. \
[eth_getFilterLogs](/arbitrum-api/eventlogs/eth_getFilterLogs) : Fetches logs matching a specific filter.

These methods are essential for tracking smart contract events in real-time on the Arbitrum network.
