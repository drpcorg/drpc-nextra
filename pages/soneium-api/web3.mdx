## Web3

Provides general network-related data, including client version details and cryptographic functions.

[web3_clientVersion](/soneium-api/web3/web3_clientVersion): Returns the software version of the connected node. \
[web3_sha3](/soneium-api/web3/web3_sha3): Generates a Keccak-256 hash for the given input data.
