## Event logs

Monitors logs from smart contracts, making it useful for tracking events and state changes.

[eth_getLogs](/soneium-api/eventlogs/eth_getLogs): Fetches logs matching a given filter. \
[eth_newFilter](/soneium-api/eventlogs/eth_newFilter): Establishes a filter to track contract events. \
[eth_getFilterChanges](/soneium-api/eventlogs/eth_getFilterChanges): Returns new logs detected since the last check. \
[eth_uninstallFilter](/soneium-api/eventlogs/eth_uninstallFilter): Removes an event filter. \
[eth_getFilterLogs](/soneium-api/eventlogs/eth_getFilterLogs): Retrieves logs matching a specified filter. \
