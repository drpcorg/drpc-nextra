## Mining

Provides data related to the mining status and rewards of a node.

[eth_coinbase](/soneium-api/mining/eth_coinbase): Returns the address of the miner currently receiving rewards. \
[eth_mining](/soneium-api/mining/eth_mining): Indicates whether the node is actively mining new blocks.
