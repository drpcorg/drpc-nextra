## Web3


Retrieves general network configuration details, including client version and cryptographic utilities.

[web3_clientVersion](/viction-api/web3/web3_clientVersion): Returns the client software version. \
[web3_sha3](/viction-api/web3/web3_sha3): Computes the Keccak-256 hash of the given data.

These methods are essential for node management and cryptographic operations on the Viction network.
