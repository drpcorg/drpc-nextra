## Event logs

Provides logs of events emitted by smart contracts, essential for tracking token transfers and state changes.

[eth_getLogs](/viction-api/eventlogs/eth_getLogs): Retrieves event logs based on filter criteria. \
[eth_newFilter](/viction-api/eventlogs/eth_newFilter): Creates a new filter to monitor specific events. \
[eth_getFilterChanges](/viction-api/eventlogs/eth_getFilterChanges): Fetches new event logs since the last filter update. \
[eth_uninstallFilter](/viction-api/eventlogs/eth_uninstallFilter): Removes an existing event filter. \
[eth_getFilterLogs](/viction-api/eventlogs/eth_getFilterLogs): Returns all past logs that match a given filter.

These methods are essential for tracking smart contract activity on Viction.
