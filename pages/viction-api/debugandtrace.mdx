## Debug and trace

Available only on the paid tier. Allows tracing transactions, inspecting block execution, and replaying transactions or blocks for detailed analysis.

[trace_filter](/viction-api/debugandtrace/trace_filter): Filters trace data based on specified criteria. \
[trace_rawTransaction](/viction-api/debugandtrace/trace_rawTransaction): Replays a raw transaction for debugging purposes. \
[trace_block](/viction-api/debugandtrace/trace_block): Traces the execution of a block. \
[trace_replayBlockTransactions](/viction-api/debugandtrace/trace_replayBlockTransactions): Replays transactions within a block for analysis. \
[trace_replayBlockTransactions#vmTrace](/viction-api/debugandtrace/trace_replayBlockTransactionsvmTrace): Provides a VM-level execution trace during block replay. \
[debug_traceBlockByHash](/viction-api/debugandtrace/debug_traceBlockByHash): Traces block execution using its hash. \
[debug_traceBlockByNumber](/viction-api/debugandtrace/debug_traceBlockByNumber): Traces block execution by its number. \
[trace_transaction](/viction-api/debugandtrace/trace_transaction): Traces a specific transactionâ€™s execution. \
[debug_traceTransaction](/viction-api/debugandtrace/debug_traceTransaction): Provides a detailed execution trace for a transaction. \
[trace_replayTransaction](/viction-api/debugandtrace/trace_replayTransaction): Replays a transaction to analyze its behavior. \
[trace_replayTransaction#vmTrace](/viction-api/debugandtrace/trace_replayTransactionvmTrace): Offers a VM-level execution trace during transaction replay. \
[trace_callMany](/viction-api/debugandtrace/trace_callMany): Executes multiple trace calls simultaneously. \
[trace_get](/viction-api/debugandtrace/trace_get): Retrieves trace data for an execution. \
[trace_call](/viction-api/debugandtrace/trace_call) : Runs a trace call without modifying state.\
[debug_traceCall](/viction-api/debugandtrace/debug_traceCall): Traces a specific contract call for debugging.

These methods offer powerful debugging and analysis tools for investigating execution flows and troubleshooting issues on Viction.
