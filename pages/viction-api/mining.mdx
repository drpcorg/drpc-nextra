## Mining

Retrieves details about the mining state of a Viction node.

[eth_coinbase](/viction-api/mining/eth_coinbase): Returns the address receiving mining rewards. \
[eth_mining](/viction-api/mining/eth_mining): Indicates whether the node is actively mining.

These methods help monitor the mining status of a Viction node.
