## Node info

Provides data related to the configuration and health of the Solana nodes, which form the backbone of the network.

[getClusterNodes](/solana-api/nodeinfo/getClusterNodes) : Retrieves the list of nodes in the cluster. \
[getHealth](/solana-api/nodeinfo/getHealth) : Checks the health status of the node. \
[getVersion](/solana-api/nodeinfo/getVersion) : Returns the version of the Solana node software. \
[getIdentity](/solana-api/nodeinfo/getIdentity) : Provides the node's identity information.

These methods are essential for monitoring the health and configuration of Solana nodes.
