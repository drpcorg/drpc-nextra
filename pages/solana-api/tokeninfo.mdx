## Token info

Focuses on accounts and balances associated with specific tokens.

[getTokenSupply](/solana-api/tokeninfo/getTokenSupply) : Retrieves the total supply of a specific token.  \
[getTokenAccountBalance](/solana-api/tokeninfo/getTokenAccountBalance) : Returns the balance of a specific token account.  \
[getTokenAccountsByDelegate](/solana-api/tokeninfo/getTokenAccountsByDelegate) : Lists token accounts associated with a delegate.  \
[getTokenAccountsByOwner](/solana-api/tokeninfo/getTokenAccountsByOwner) : Fetches token accounts owned by a specific address.  \
[getTokenLargestAccounts](/solana-api/tokeninfo/getTokenLargestAccounts) : Provides a list of the largest token accounts.  \
[requestAirdrop](/solana-api/tokeninfo/requestAirdrop) : Requests an airdrop of tokens to a specified account.

These methods are essential for managing token-related operations on the Solana network.
