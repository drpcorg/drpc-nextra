## Network info

Offers aggregate data related to the overall network status and operational metrics.

[getEpochInfo](/solana-api/networkinfo/getEpochInfo) : Retrieves information about the current epoch.  \
[getEpochSchedule](/solana-api/networkinfo/getEpochSchedule) : Provides the schedule for the current epoch.  \
[getFeeForMessage](/solana-api/networkinfo/getFeeForMessage) : Returns the fee for processing a specific message.  \
[getHighestSnapshotSlot](/solana-api/networkinfo/getHighestSnapshotSlot) : Fetches the highest snapshot slot in the network.  \
[getGenesisHash](/solana-api/networkinfo/getGenesisHash) : Retrieves the hash of the genesis block.  \
[getRecentPerformanceSamples](/solana-api/networkinfo/getRecentPerformanceSamples) : Provides recent performance metrics for the network.  \
[getFirstAvailableBlock](/solana-api/networkinfo/getFirstAvailableBlock) : Returns the first available block in the chain.  \
[getMinimumBalanceForRentExemption](/solana-api/networkinfo/getMinimumBalanceForRentExemption) : Fetches the minimum balance required for rent exemption. \
[getFees](/solana-api/networkinfo/getFees) : Returns the current fee structure for transactions.  \
[getFeeCalculatorForBlockhash](/solana-api/networkinfo/getFeeCalculatorForBlockhash) : Provides the fee calculator for a specific blockhash.  \
[getRecentPrioritizationFees](/solana-api/networkinfo/getRecentPrioritizationFees) : Retrieves recent prioritization fees for transactions.

These methods are essential for monitoring the operational status of the Solana network.
