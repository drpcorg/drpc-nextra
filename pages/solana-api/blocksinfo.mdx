## Blocks info

Provides detailed insights about blocks on the blockchain.

[getBlockCommitment](/solana-api/blocksinfo/getBlockCommitment) : Fetches the commitment level of a block. \
[getBlockProduction](/solana-api/blocksinfo/getBlockProduction) : Provides information on block production rates. \
[getBlocks](/solana-api/blocksinfo/getBlocks) : Lists blocks based on specified criteria. \
[getBlock](/solana-api/blocksinfo/getBlock) : Retrieves details of a specific block. \
[getBlocksWithLimit](/solana-api/blocksinfo/getBlocksWithLimit) : Fetches a limited number of blocks. \
[getBlockHeight](/solana-api/blocksinfo/getBlockHeight) : Returns the height of a specific block. \
[getBlockTime](/solana-api/blocksinfo/getBlockTime) : Provides the timestamp for a block. \
[getConfirmedBlocks](/solana-api/blocksinfo/getConfirmedBlocks) : Lists confirmed blocks in the chain. \
[getConfirmedBlock](/solana-api/blocksinfo/getConfirmedBlock) : Retrieves a confirmed block's details. \
[isBlockhashValid](/solana-api/blocksinfo/isBlockhashValid) : Checks if a blockhash is valid. \
[getLatestBlockhash](/solana-api/blocksinfo/getLatestBlockhash) : Returns the latest blockhash.

These methods provide essential insights for analyzing blockchain data on the Solana network.
