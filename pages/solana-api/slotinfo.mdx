## Slot info

Slots represent periods of time during which a leader is responsible for processing transactions. These methods provide data related to slot timing and leadership.

[getMaxRetransmitSlot](/solana-api/slotinfo/getMaxRetransmitSlot) : Retrieves the maximum slot for retransmission. \
[getMaxShredInsertSlot](/solana-api/slotinfo/getMaxShredInsertSlot) : Provides the maximum slot for shred insertion. \
[getSlot](/solana-api/slotinfo/getSlot) : Returns the current slot. \
[getSlotLeader](/solana-api/slotinfo/getSlotLeader) : Fetches the leader for the current slot. \
[getSlotLeaders](/solana-api/slotinfo/getSlotLeaders) : Lists leaders for upcoming slots. \
[minimumLedgerSlot](/solana-api/slotinfo/minimumLedgerSlot) : Retrieves the minimum ledger slot. \
[getLeaderSchedule](/solana-api/slotinfo/getLeaderSchedule) : Provides the schedule of slot leaders.

These methods help monitor and analyze slot-related data on the Solana network.
