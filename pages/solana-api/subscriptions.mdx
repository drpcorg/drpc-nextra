## Subscriptions

Allows clients to receive real-time updates and notifications about various on-chain events.

[accountSubscribe](/solana-api/subscriptions/accountSubscribe) : Subscribes to account updates.  \
[accountUnsubscribe](/solana-api/subscriptions/accountUnsubscribe) : Unsubscribes from account updates.  \
[blockSubscribe](/solana-api/subscriptions/blockSubscribe) : Subscribes to block updates.  \
[blockUnsubscribe](/solana-api/subscriptions/blockUnsubscribe) : Unsubscribes from block updates.  \
[logsSubscribe](/solana-api/subscriptions/logsSubscribe) : Subscribes to log updates.  \
[logUnsubscribe](/solana-api/subscriptions/logUnsubscribe) : Unsubscribes from log updates.  \
[programSubscribe](/solana-api/subscriptions/programSubscribe) : Subscribes to program updates.  \
[programUnsubscribe](/solana-api/subscriptions/programUnsubscribe) : Unsubscribes from program updates.  \
[signatureSubscribe](/solana-api/subscriptions/signatureSubscribe) : Subscribes to signature updates.  \
[signatureUnsubscribe](/solana-api/subscriptions/signatureUnsubscribe) : Unsubscribes from signature updates.  \
[slotSubscribe](/solana-api/subscriptions/slotSubscribe) : Subscribes to slot updates.  \
[slotUnsubscribe](/solana-api/subscriptions/slotUnsubscribe) : Unsubscribes from slot updates.

These methods enable real-time monitoring of important events on the Solana network.
