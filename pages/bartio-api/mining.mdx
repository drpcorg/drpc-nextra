## Mining

Delivers information about the node's mining activities.

[eth_coinbase](/bartio-api/mining/eth_coinbase) : Fetches the address assigned to receive mining rewards. \
[eth_mining](/bartio-api/mining/eth_mining) : Indicates whether the node is actively engaged in mining new blocks.

These endpoints are essential for tracking and managing the mining operations of a node on the Berachain network.
